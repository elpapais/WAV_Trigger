WAV Trigger
===============

[![WAV Trigger](.jpg)  
*WAV Trigger (BOB-xxxxx)*]()

The WAV Trigger is similar to its cousin, the MP3 Trigger, however it plays .WAV files and allows them to be mixed on the fly.
